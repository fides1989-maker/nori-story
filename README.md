# Нори. Наша история

Интерактивная история-игра на день рождения. Vanilla HTML/CSS/JS, без сборки и зависимостей.

## Как открыть

Открыть `index.html` в любом современном браузере или открыть live-версию (см. ниже).

## Уровни

1. **Печатай.ру** — найти 5 ошибок на макете визитки
2. **Бизнес в зелёной комнате** — три бизнес-ветки с выборами
3. **Своё гнездо. Карантин** — стелс-аркада с QR + поиск 10 предметов для Дня Грузии
4. **Банк. Делегатор 3000** — кликер с делегированием задач, шкалами усталости и режимом «🐾 Лапки»
5–7. В разработке

## Деплой

Открыт через GitHub Pages — см. вкладку Settings → Pages в репозитории.
